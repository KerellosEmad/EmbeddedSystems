# C1M2 - Makefile Build System Project

## Description
This project demonstrates how to build a modular C project using GNU Make and an ARM embedded toolchain.

---

## 📁 Project Structure

- Makefile → Build automation script
- sources.mk → Source file definitions
- include/ → Header files
- src/ → Source code files
- msp432p401r.lds → Linker script

---

## ⚙️ Build Instructions

Build the project:
make

Clean build files:
make clean

---

## Learning Goals

- Using GNU Make
- Multi-file C projects
- Embedded build systems
- ARM GCC toolchain usage
