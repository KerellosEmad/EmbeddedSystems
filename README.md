# Statistical Analysis Project (C Programming)

## Author
Kerellos Emad Yani  

## Description
This project is part of the Coursera Embedded Systems course (C1M1 assignment).  
It implements statistical analysis on an array of unsigned char data.

The program performs:
- Minimum value calculation
- Maximum value calculation
- Mean (average) calculation
- Median calculation
- Sorting the array in descending order
- Printing the array and statistics

## Files Included
- stats.c → Implementation of all statistical functions and main program
- stats.h → Function declarations and documentation
- README.md → Project description

## How to Compile
```bash
gcc stats.c -o stats.out
